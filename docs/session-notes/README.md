## 📚 How to Use This Folder

### Quick Create (1 minute)

1. Copy `_TEMPLATE.md`
2. Rename to `YYYY-MM-DD-topic.md`
3. Fill sections 1, 2, 4, 5 (minimum)
4. Add one row to the archive table below

---

### When to Create a Session Note

Create a new session note when you:
- Make important architectural decisions
- Discuss new feature designs
- Plan multi-phase initiatives
- Solve complex technical problems
- Review project status

### Naming Convention

Format: `YYYY-MM-DD-topic-name.md`

Examples:
- `2026-03-01-strategic-planning.md`
- `2026-03-15-mcp-integration-poc.md`
- `2026-04-10-performance-optimization.md`
